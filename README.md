# Avoryn

Avoryn is a mobile app concept for smarter everyday decisions.

This repository starts with a simple Expo + React Native + TypeScript foundation so the app can run in Expo Go first.

## Run locally

```bash
npm install
npm start
```

Then scan the QR code with Expo Go.

## Current v0.1 goal

- Open the app in Expo Go
- Show a simple Avoryn home screen
- Add tab navigation: Ask, Nearby, Scan, Profile
- Use a clean icon pack through `@expo/vector-icons`
